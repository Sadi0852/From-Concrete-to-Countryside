# From-Concrete-to-Countryside

Computer Graphics Project (OpenGL + C++)

A scene-based animated graphics project developed using C++ and OpenGL (GLUT) for the Computer Graphics course.

The project visually represents the transformation “From Concrete to Countryside”, demonstrating graphical rendering, animation, interaction, and scene transitions using OpenGL.

## Project Overview

This project showcases:

 -2D graphical scene rendering

 -Animation effects

 -Day and night transitions

 -Interactive keyboard controls

 -Sound integration

 -Scene-based storytelling

It demonstrates practical implementation of core Computer Graphics concepts using OpenGL.

## Controls & Interaction

| Key           | Function         |
| ------------- | ---------------- |
| `Space`       | Stop movement    |
| `S`           | Restart movement |
| `Right Arrow` | Day mode         |
| `Left Arrow`  | Night mode       |
